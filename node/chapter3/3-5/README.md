# node 내장모듈 사용

## os
운영체제 정보를 가져올 수 있는 모듈

## path
디렉토리 / 파일의 경로를 쉽게 조작하도록 도와주는 모듈  
windows: a\b\c\d 로 구분 ( \ )  
POSIX: a/b/c/d 로 구분 ( / )  

## url
인터넷 주소를 쉽게 조작할 수 있도록 도와주는 모듈  
WHATWG의 url 구분방법을 따름  

## querystring
WHATWG방식의 url 대신 node 방식의 url을 사용할때  
search 부분을 사용하기 쉽게 객체로 만들어주는 모듈

## crypto
암호화 모듈

## util
편의모듈
