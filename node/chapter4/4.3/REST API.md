# REST API
REST API에서 사용되는 HTTP Request

### GET
서버 리소스를 가져올때 사용.  
body에 데이터를 넣지 않고, 필요에 따라 QueryString으로 데이터 전달

### POST

서버에 새로운 리소스를 등록 할 때 사용.  
body에 리소스 데이터를 담음.

### PUT

서버의 리소스를 변경할 때 사용.  
body에 변경 할 리소스 데이터를 담음.

### PATCH

서버 리소스의 일부만 변경할 때 사용.  
body에 변경 할 데이터를 담음.

### DELETE

서버 리소스 삭제할 때 사용.
